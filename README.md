# Cookie Bookie

## Description
Cookie Bookie is a lightweight, client-side web app for managing cookie recipes. It offers a clean, responsive UI with full CRUD functionality. Users can add, update, and delete recipes, with instant feedback and localStorage persistence. Each recipe includes a title, optional image URL, ingredients, and steps.

## Features
- **Create, Read, Update, Delete recipes
- **Local persistence via localStorage
- **Search recipes by title or ingredientsed card flips.
- **Seed example recipes for quick demos.
- **Inline editing with a dedicated edit mode.
- **Responsive layout with a two-column home page on larger screens.
- **Client-side validation to ensure required fields.

## Tech Stack
- HTML, CSS, JavaScript (ES6+)
- Browser APIs: Canvas, localStorage, timers, events
- Modular JavaScript with classes and reusable functions
